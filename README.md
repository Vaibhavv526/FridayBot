# 💬 Currency Conversion Chatbot

A chatbot that understands natural language and performs real-time currency conversion.

Built using Dialogflow for NLP, Flask for backend processing, and deployed on Render.

---

## 🚀 Features

- 💬 Natural language interaction  
- 💱 Real-time currency conversion  
- 🌍 Supports multiple currencies  
- ⚡ Fast webhook-based responses  

---

## 🛠️ Tech Stack

- Dialogflow (NLP)
- Flask (Python backend)
- CurrencyAPI (exchange rates)
- Render (deployment)
- ngrok (testing)

---

## 🧠 How It Works

User → Dialogflow → Flask Webhook → CurrencyAPI → Response → User

- Dialogflow extracts amount and currencies  
- Flask processes the request  
- CurrencyAPI provides exchange rates  
- Result is returned to the user
